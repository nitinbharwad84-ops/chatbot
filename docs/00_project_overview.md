# Project Overview: AI SaaS Chatbot

## Vision
To build a premium, high-performance AI Chatbot SaaS using the Next.js + Groq + Supabase stack. The application focuses on speed (sub-500ms streaming), a modular AI model selection system, and a clean, modern UI inspired by industry leaders like Claude and ChatGPT.

## North Star Metrics
- **Performance:** Stream latency < 500ms.
- **Reliability:** 99.9% Auth & Message success rate.
- **UX:** Zero-friction multi-chat management.

## Core Tech Stack
- **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS, Framer Motion.
- **Backend:** Next.js Edge/API Routes.
- **Database:** Supabase (PostgreSQL).
- **Auth:** Supabase Auth (Email/Password).
- **AI:** Groq API (High-speed Inference).

## AI Models Supported
- `openai/gpt-oss-120b`
- `qwen/qwen3-32b`
- `llama-3.3-70b-versatile`
- `llama-3.1-8b-instant`
